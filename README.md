# Desafio Café Brasil: Sistemas Cafeeiros e Conexão Embrapa

Quiz educativo e interativo sobre sistemas cafeeiros, ATER+Digital Café, publicações e soluções tecnológicas da Embrapa.

## Sobre

O projeto apresenta 32 perguntas sobre:

- Conhecendo o café
- Território, clima e adaptação
- Genética, cultivares e planejamento
- Solo, água e formação da lavoura
- Manejo produtivo e fitossanidade
- Pós-colheita, qualidade e cafés especiais
- Torra, consumo e compostos do café
- Sustentabilidade, mercado e futuro

Cada resposta inclui uma explicação curta e uma seção **Conexão Embrapa**, com links para ATER+Digital, publicações e soluções tecnológicas quando disponíveis. A lógica editorial prioriza uma visão integrada de diversidade, território, genética, manejo, qualidade, sustentabilidade e inovação.

## Como rodar localmente

Abra `index.html` diretamente no navegador ou rode um servidor local:

```bash
python -m http.server 4173
```

Depois acesse:

```text
http://127.0.0.1:4173/
```

## Publicação no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie estes arquivos para a branch `main`.
3. No GitHub, entre em `Settings > Pages`.
4. Em `Build and deployment`, selecione:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Salve e aguarde a URL pública do GitHub Pages.
